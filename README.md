

  
## Environment

# Installation
### ROS2 Humble

I used this script

```shell
https://github.com/Tiryoh/ros2_setup_scripts_ubuntu
```

### download source code
```shell
git clone https://github.com/MJU-Capstone-PetRobot/mjbot_2023.git
```

### pip install
```shell
pip install -r requirements.txt
```

### run ./setup.sh

```shell
./setup.sh
```
